# OpenAssetWatch Agent Debian Package Source

This directory contains Debian package-owned source files for
`openassetwatch-agent`.

The Debian package installs the root-owned agent binary under
`/opt/openassetwatch/agent/bin/oaw-agent`, installs systemd units under
`/lib/systemd/system`, installs root-owned privileged helpers under
`/usr/lib/openassetwatch/agent/libexec`, and installs a narrow sudoers
allowlist under `/etc/sudoers.d/openassetwatch-agent`.

Real `config.json` and `identity.json` files are administrator-managed and are
not generated by the package.
