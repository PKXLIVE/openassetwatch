# OpenAssetWatch Agent Debian Package Source

This directory contains Debian package-owned source files for
`openassetwatch-agent`.

The Debian package installs the root-owned agent binary under
`/opt/openassetwatch/agent/bin/oaw-agent`, installs systemd units under
`/lib/systemd/system`, installs root-owned privileged helpers under
`/usr/lib/openassetwatch/agent/libexec`, and installs a narrow sudoers
allowlist under `/etc/sudoers.d/openassetwatch-agent`.

Real `config.json` and `identity.json` files are administrator-managed and are
not generated by the package.

The committed maintainer scripts are the approved lifecycle source:

- `postinst` validates or creates the non-interactive `openassetwatch` service
  principal, reloads systemd when active, enables `oaw-agent.timer`, and
  restarts the timer only when real config and identity files already exist.
- `prerm` stops the timer and any active oneshot service before upgrade or
  final removal, disables the timer on final removal, and removes only the
  OpenAssetWatch timer enablement symlink.
- `postrm` performs daemon-reload cleanup only and preserves config, identity,
  state, logs, and the service principal by default.
