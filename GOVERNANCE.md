# OpenAssetWatch Governance

OpenAssetWatch is maintained as an open-source project.

## Official repository

The official source repository is:

https://github.com/PKXLIVE/openassetwatch

## Maintainer authority

Maintainers control:

- Repository settings
- Release publication
- Security advisories
- Package signing
- Roadmap acceptance
- Contributor access

## Decision making

Routine changes may be merged by maintainers after review and successful checks.

Security-sensitive, legal, branding, release, installer, or governance changes require maintainer approval.

## Releases

Only maintainers may publish official OpenAssetWatch releases. Official releases must come from this repository or another repository explicitly listed by the maintainers.

## Forks

Forks are allowed under the project license, but forks must not misrepresent themselves as the official OpenAssetWatch project.
