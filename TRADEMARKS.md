# OpenAssetWatch Trademark and Brand Policy

OpenAssetWatch is the name of this open-source project and its official release artifacts.

## Permitted use

You may use the OpenAssetWatch name to:

- Refer to the official project
- Describe compatibility with OpenAssetWatch
- Identify that your work is a fork or derivative
- Provide attribution required by the Apache License, Version 2.0

## Not permitted

You may not use the OpenAssetWatch name, logo, branding, package names, release names, or confusingly similar marks to:

- Imply your fork is the official project
- Publish unofficial releases as official OpenAssetWatch releases
- Mislead users about maintainership, endorsement, or origin
- Distribute modified installers without clearly marking them as modified
- Operate a repository, package, domain, or service that impersonates the official project

## Fork naming

Forks should use a clearly distinct name unless they are only hosted as normal GitHub forks.

## Official project locations

The maintainers may publish official locations in the README, release notes, documentation, and package metadata.
