# Quarantined Legacy/Source Material

This directory is reserved for tool YAMLs, screenshots, notes, or other material
that came from a different/source project and is unsafe or not yet reviewed for
OpenAssetWatch.

Files in this directory are not active OpenAssetWatch tools. They are kept only
for review and reference. OpenAssetWatch loaders, installers, and production
configuration paths must not load anything from `configs/quarantine/`.

Unsafe or offensive tools must stay quarantined unless they are rewritten into a
defensive, passive-first, evidence-based OpenAssetWatch pattern. Quarantined
material must not include live secrets.

Initial review result for this pass: no active legacy/source project tool YAMLs
were found in the repository. This directory is being added as a guardrail for
future review work.
